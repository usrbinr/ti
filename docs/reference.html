Function Details

ytd()

Calculate Year-to-Date aggregation.

Usage

ytd(.data, .date, .value, calendar_type, fiscal_year_start = 1)

Arguments

Argument Description
.data A tibble or lazy DBI table
.date The date column to index
.value The value column to aggregate
calendar_type Either "standard" or retail calendar ("445", "454", "544")
fiscal_year_start Month (1-12) when fiscal year begins (default: 1)

abc()

Perform ABC Classification (Pareto analysis) on grouped data.

Usage

abc(.data, category_values, .value)

Arguments

Argument Description
.data A grouped tibble or lazy DBI table
category_values Cumulative percentage breakpoints (e.g., c(0.8, 0.95, 1) for 80/95/100)
.value The value column to classify on

cohort()

Perform cohort analysis to track entities over time.

Usage

cohort(.data, .date, .value, time_unit = "month", period_label = FALSE)

Arguments

Argument Description
.data A tibble or lazy DBI table
.date The date column for time periods
.value The entity identifier column (e.g., customer_key)
time_unit Time unit for cohorts ("day", "week", "month", "quarter", "year")
period_label Whether to include period labels (default: FALSE)
